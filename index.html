What is Project XPlor?

Project XPlor is an app that allows

users to store travel information and data

It also gives user access to a timeline

displaying the differnt sections for each trip,

as well as any information regarding it as well.


The aim of this app is to make travel organization

simple, keeping importnat information easy to

access. With the design of the app, you are able

to plan the tip you have always been dreaming about!
